sets next condition (if any) employing the following formula\+:

f(i,j) = i / prod\+\_\+\{k=0\}$^\wedge$\{j-\/1\} (ck) mod cj

where i = combination index j = random variable index cj = j-\/th variable cardinality (number of parent states) prod\+\_\+\{k=0\}$^\wedge$\{j-\/1\} is the the product for k = \{0, 1, ..., j-\/1\}

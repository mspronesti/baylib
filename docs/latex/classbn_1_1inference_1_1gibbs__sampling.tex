===== Gibbs sampling Algorithm ===

This class represents the Gibbs Sampling approximate inference algorithm for discrete Bayesian Networks. It\textquotesingle{}s based on the Gibbs sampler. It offers the possibility to use a custom generator and an initial seed NOTICE\+: Gibbs sampling should not be used for bayesian networks with deterministic nodes, i.\+e. nodes with some entry in the cpt equal to 1.\+0

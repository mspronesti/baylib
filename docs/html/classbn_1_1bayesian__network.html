template<typename Probability_>

This class models a Bayesian Network_ allowing both index and name based access to its facilities for a better user experience

template<typename Probability_ >
void bn::bayesian_network< Probability_ >::optimize_cpt_memory_occupation ( vertex_id  id)
inlineprivate

Checks if a specific variable, identified by numerical identifier, has a cpt that is already present in the network, if it's the case the cpt of the node is deleted from memory and switched with a reference to the one that already exists, the cpt will be copied if a write operation is executed following the COW paradigm

Parameters
id: numerical identifier of node

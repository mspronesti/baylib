Detailed Description

template<typename Probability_ = double, typename Generator_ = std::mt19937>
class bn::inference::likelihood_weighting< Probability_, Generator_ >

===== Likelihood Weighting Algorithm ===

This class represents the likelihood-weighting approximate inference algorithm for discrete Bayesian Networks. It offers the possibility to use a custom generator and an initial seed

Template Parameters
Probability_: the type expressing the probability
Generator_: the random generator (default Mersenne Twister pseudo-random generator)

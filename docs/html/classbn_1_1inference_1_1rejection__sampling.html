Detailed Description

template<typename Probability_ = double, typename Generator_ = std::mt19937>
class bn::inference::rejection_sampling< Probability_, Generator_ >

========== Rejection Sampling Algorithm ========= This class represents the rejection sampling approximate inference algorithm for discrete bayesian networks. It allows to specify a custom generator and a custom initial seed. Please NOTICE: to use a non-standard generator you must overload

  • min()
  • max()
  • operator () as they're required by std::discrete_distribution
    Template Parameters
    Probability_: the type expressing the probability
    Generator_: the random generator (default Mersenne Twister pseudo-random generator)

Detailed Description

template<BNetDerived Network_, typename Generator_ = std::mt19937>
class bn::inference::rejection_sampling< Network_, Generator_ >

========== Rejection Sampling Algorithm ========= This class represents the rejection sampling approximate inference algorithm for discrete bayesian networks. It allows to specify a custom generator and a custom initial seed. Please NOTICE: to use a non-standard generator you must overload

  • min()
  • max()
  • operator () as they're required by std::discrete_distribution
    Template Parameters
    Network_: the type of bayesian network (must inherit from bn::bayesian_network)
    Generator_: the type of random generator (default Mersenne Twister pseudo-random generator)

Detailed Description

template<BNetDerived Network_, typename Generator_ = std::mt19937>
class bn::inference::logic_sampling< Network_, Generator_ >

This class represents the Logic Sampling approximate inference algorithm for discrete Bayesian Networks. The implementation uses boost::compute to exploit GPGPU optimization. All samples are simulated in parallel, for this reason the maximum memory usage tolerable must be specified to avoid filling up the memory of the device in case of large number of samples. The algorithm main steps are :

  1. sort nodes in topological order
  2. simulate each source node
  3. simulate each child node given the parents previous simulation
  4. throw out each simulation that doesn't comply with observed evidences
  5. estimate the marginal distribution from the valid simulations
    Template Parameters
    Network_: the type of bayesian network (must inherit from bn::bayesian_network)
    Generator_: the type of random generator (default Mersenne Twister pseudo-random generator)

template<typename Probability_ >
bool bn::condition_factory< Probability_ >::has_next ( )
inline

sets next condition (if any) employing the following formula:

f(i,j) = i / prod_{k=0}^{j-1} (ck) mod cj

where i = combination index j = random variable index cj = j-th variable cardinality (number of parent states) prod_{k=0}^{j-1} is the the product for k = {0, 1, ..., j-1}

Returns
true if new condition false otherwise

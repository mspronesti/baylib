Detailed Description

template<Arithmetic Probability_ = double>
class baylib::named_random_variable< Probability_ >

This class is used to support xdsl additional features associated to a random variable, inheriting from random_variable. Offers to possibility to name the variable and its states.

Template Parameters
Probability_: the type expressing the probability must be arithmetic

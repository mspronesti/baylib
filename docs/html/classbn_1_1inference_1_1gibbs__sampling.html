Detailed Description

template<typename Probability = double, typename Generator = std::mt19937>
class bn::inference::gibbs_sampling< Probability, Generator >

===== Gibbs sampling Algorithm ===

This class represents the Gibbs Sampling approximate inference algorithm for discrete Bayesian Networks. It's based on the Gibbs sampler. It offers the possibility to use a custom generator and an initial seed NOTICE: Gibbs sampling should not be used for bayesian networks with deterministic nodes, i.e. nodes with some entry in the cpt equal to 1.0

Template Parameters
Probability: the type expressing the probability
Generator: the random generator (default Mersenne Twister pseudo-random generator)

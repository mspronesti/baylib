Detailed Description

template<BNetDerived Network_>
class baylib::condition_factory< Network_ >

this class produces all the combinations for a given random variable of a bayesian network to fill its conditional probability table

Template Parameters
Network_: the type of network the factory works on (must inherit from baylib::bayesian_net)

◆ has_next()

template<BNetDerived Network_>
bool baylib::condition_factory< Network_ >::has_next ( )
inline

sets next condition (if any) employing the following formula:

f(i,j) = i / prod_{k=0}^{j-1} (ck) mod cj

where i = combination index j = random variable index cj = j-th variable cardinality (number of parent states) prod_{k=0}^{j-1} is the the product for k = {0, 1, ..., j-1}

Returns
true if new condition false otherwise

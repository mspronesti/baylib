Detailed Description

template<BNetDerived Network_>
class bn::inference::vectorized_inference_algorithm< Network_ >

This class models an approximate inference algorithm vectorized with a GPGPU approach. the method simulate_node samples a node given the results of previous simulations of its parents nodes

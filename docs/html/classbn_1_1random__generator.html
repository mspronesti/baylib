Detailed Description

template<typename Tp_, typename Generator_ = std::mt19937>
class bn::random_generator< Tp_, Generator_ >

Random generator wrapper for sampling-based inference algorithms Offered customization capabilities:

  • custom generator
  • custom seed
  • custom type for the random number
    Template Parameters
    Tp_: the type of the random number
    Generator_: the type of generator (default Marsenne Twister)
